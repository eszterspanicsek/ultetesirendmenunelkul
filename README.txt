Esküvői QR ültetési rend – v4

Mi van a csomagban?
- admin.html -> szerkesztő felület
- index.html -> külön vendégoldal
- data.js -> a vendéglista és a szövegek

Használat:
1. Csomagold ki a ZIP-et.
2. Nyisd meg az admin.html fájlt Chrome-ban.
3. Írd át a szövegeket, színeket és vendéglistát.
4. Nyomj Mentés.
5. A jobb oldali előnézet azonnal frissül.

Letöltési lehetőségek az adminban:
- data.js letöltése: ha a teljes csomagot akarod később feltölteni webre
- Kész vendégoldal letöltése: egyetlen HTML fájl, amit külön is használhatsz

Fontos:
- A Mentés a böngészőben tárolja az adatokat ugyanazon az eszközön.
- A QR-kódos éles használathoz ajánlott a kész vendégoldalt vagy a teljes csomagot feltölteni egy webcímre.
- Ez teljesen különálló az esküvői weboldalatoktól.
